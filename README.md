# Academic Portfolio

A simple, modern academic portfolio website to showcase research, publications, and academic achievements.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Start development server: `pnpm dev`
4. Open `http://localhost:5173`

## Customization

Edit `src/data/portfolioData.ts` with your personal information, education, publications, and experience. The website will automatically update with your content.

## Building for Production

```bash
pnpm build
```
